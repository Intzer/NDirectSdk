# [nDirect](https://ndirect.ru) SDK for Monitoring Integrations

Легковесная и производительная PHP-библиотека для интеграции мастер-сервера **[nDirect](https://ndirect.ru)** с мониторингами. SDK обеспечивает быструю отдачу списков серверов (Search, GameMenu, Favourites), получение детальной статистики и её визуализацию с использованием кэширования.


## Основные возможности

- **Поддержка различных движков:** архитектура позволяет подключать любые мониторинги (по умолчанию реализована полная интеграция для **SVV**).
    
- **Кэширование:** Использование **SQLite** для хранения ответов API.
    
- **Гибкая сортировка:** Поддержка рандомизации выдачи или жесткой сортировки по реальному количеству коннектов.
    
- **Готовые графики:** Встроенный механизм рендеринга графиков (установки, уникальные игроки, удаления, коннекты).
    
## Требования

- **PHP:** 7.0 или выше.
    
- **Расширения:** `curl`, `pdo_sqlite`, `pdo_mysql`, `json`.
    
- **Права доступа:** Папка с файлом кэша (`cache_file`) должна быть доступна для записи веб-серверу (права 755/777).


## Установка и настройка

1. Разместите папку `[nDirect](https://ndirect.ru)Sdk` в директории вашего мониторинга.
    
2. Настройте параметры подключения и ключи в файле `config.php`.


## Быстрый старт

Самый простой способ получить доступ к функционалу в любой части вашего кода - использовать Singleton-клиент:

```php
require_once 'path/to/NDirectSdk/src/Autoloader.php';

// Получение инстанса интеграции
$sdk = \NDirectSdk\NDirectClient::getInstance()->get();

// Пример: получение коннектов за сегодня и вчера
echo $sdk->getConnects(1)
;
```

## Получение данных

Вы можете запрашивать различные метрики за определенный период в днях (нужно указывать число на 1 меньше):

```php
$unique = $sdk->getUnique(6);      // Уникальные игроки за неделю
$installs = $sdk->getInstalls(6);    // Установки за неделю
$connects = $sdk->getConnects(1);    // Коннекты к серверам за сегодня и вчера
$uninstalls = $sdk->getUninstalls(7);  // Удаления за неделю
```


## Рендеринг графика

Метод `renderChart` подставляет данные в шаблон и возвращает готовый HTML/JS блок:

```php
// В вашем PHP скрипте
echo $sdk->renderChart($sdk->getUnique(6));
```


## Эндпоинты (API)

SDK предоставляет готовый интерфейс для получения списков серверов по протоколу HTTP. Основная точка входа:

**Базовый URL:** `домен/NDirectSdk/index.php`

### Получение списков серверов

Для получения списка используйте параметр `action=get-lists` и укажите тип списка через `type`.

|**Тип списка (type)**|**Описание**|**Формат ответа**|
|---|---|---|
|`search`|Основной поиск (MasterServer)|Plain text (IP:Port\n)|
|`gamemenu`|Серверы для GameMenu|Plain text (IP:Port\n)|
|`favourites`|Избранные серверы|Plain text (IP:Port\n)|

**Пример запроса:**

`https://домен/NDirectSdk/index.php?action=get-lists&type=search`

## 📊 Доступ к графикам (Iframe)

Для вставки графиков на сторонние страницы или в другие части мониторинга используйте `chart_iframe.php`:

**URL:** `домен/NDirectSdk/chart_iframe.php?type={METRIC}&days={DAYS}`

- **Метрики (`type`):** `unique` (уникальные), `installs` (установки), `connects` (коннекты), `uninstalls` (удаления).
    
- **Период (`days`):** количество дней для статистики (по умолчанию 7).
    

**Пример вставки:**

```html
<iframe src="https://домен/NDirectSdk/chart_iframe.php?type=unique&days=14" width="100%" height="400" frameborder="0"></iframe>
```

    
## Добавление новой интеграции

Чтобы добавить поддержку другого мониторинга, создайте класс в папке `Library/Integrations/`, унаследовав его от `AbstractIntegration`:

1. Реализуйте метод `getLists($type)`.
    
2. Добавьте новый `case` в `IntegrationFactory`.
